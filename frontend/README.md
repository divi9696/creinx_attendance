# Creinx Attendance System - Frontend

A React-based attendance management system frontend for tracking employee attendance with geolocation and IP validation.

## Features

- User login and authentication
- Real-time attendance marking with geolocation
- Employee attendance history
- Admin dashboard with attendance reports
- Role-based access control

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will start on `http://localhost:3000`

## Build for Production

```bash
npm build
```

## Project Structure

- `/src/components` - Reusable React components
- `/src/pages` - Page components
- `/src/styles` - CSS stylesheets

## API Integration

The frontend communicates with the backend API at `http://localhost:5000/api`

## Technologies Used

- React 18
- React Router
- Axios
- CSS3
